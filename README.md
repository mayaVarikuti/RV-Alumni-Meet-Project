# RV-Alumni-Meet-Project
## An HTML form to collect info about visitors coming to the alumni meet. Connected to a Google Sheet.
### View Google Sheet at https://docs.google.com/spreadsheets/d/1kmv3n3qZwZ7uWWV2cJCTxulukFNeEIdyCgOPcwblG8E/edit?usp=sharing.
#### To see the doPost() function, which inserts data into the spreadsheet:
#### - Go to Google Sheet.
#### - Click "Extensions," then "App Script".
#### - This should bring you to the web form handler.
